A terminal-based zombie survival quiz game built with Python. Answer 10 questions and find out how long you'd last in the apocalypse!

---

## 🛠️ Description

10 multiple-choice questions test your survival instincts — covering everything from choosing weapons to outsmarting hordes. Your score determines your fate, ranging from "eaten within the hour" to building a post-apocalyptic utopia.

---

## ⚙️ Requirements

- Python 3.x
- No external libraries needed — uses only Python's built-in `random` module

---

## 🚀 How to Run

Clone the repository and run the game:

```bash
git clone https://github.com/anjalee-29/python-mini-projects/Zombie_Game.git
cd Zombie_Game
python zombie.py
```

---

## 🎮 How to Play

- You will be asked 10 survival scenario questions
- Each question has 4 options — enter the number of your choice and press Enter
- Your score is tallied at the end to reveal your fate
- Scores include a random element, so results may vary between runs!
